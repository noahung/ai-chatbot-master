How to Use

Add this code to your website to embed the AI chatbot:

<script>
  (function() {
    const script = document.createElement('script');
    script.src = 'https://noahung.github.io/ai-chatbot-master/public/embed.js';
    script.async = true;
    script.dataset.clientId = 'YOUR_CLIENT_ID';
    script.dataset.primaryColor = '#2563eb';
    script.dataset.secondaryColor = '#ffffff';
    script.dataset.position = 'bottom-right';
    script.dataset.welcomeMessage = "Hello! How can I help you today?";
    script.dataset.placeholderText = "Ask me anything...";
    script.dataset.chatbotName = "AI Assistant";
    script.dataset.apiKey = 'YOUR_OPENAI_API_KEY';
    script.dataset.model = 'gpt-3.5-turbo';
    document.head.appendChild(script);
  })();
</script>

Configuration Options

  • clientId: Your unique client identifier
  • primaryColor: The main color for the chatbot (hex code)
  • secondaryColor: Background color for the chat window (hex code)
  • position: Where the chatbot appears on the page (bottom-right, bottom-left, top-right, top-left)
  • welcomeMessage: The initial message shown to users
  • placeholderText: Text shown in the input field
  • chatbotName: Name displayed in the chatbot header
  • apiKey: Your OpenAI API key
  • model: The OpenAI model to use (default: gpt-3.5-turbo)
